    What are keyboard shortcuts and how do I use them in Skype?

    Back to search results

    Keyboard shortcuts are specific combinations of two or more keys on your keyboard. They make it easier and faster for you to use commands that you would otherwise have to access via the menus. You can use keyboard shortcuts to easily quit Skype, send a file, copy text and a lot more.

    Skype for Windows 10 (version 14) global hotkeys

    Global hotkeys allow you to use Skype keyboard shortcuts while Skype for Windows 10 (version 14) is minimized or not in focus. Global hotkeys is enabled by default in Skype. To disable them:

    1. Click your Profile picture.
    2. Click Settings.
    3. Click General.
    4. Toggle Enable global hotkeys off.
